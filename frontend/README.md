# MediSync Healthcare Appointment & Patient Record Management System

## Project Overview

MediSync is a full-stack healthcare management platform developed using ReactJS, Node.js, Express.js, and SQLite.

The system helps clinics and hospitals manage:

- Doctor consultation slots
- Patient appointments
- Consultation records
- Healthcare analytics
- Appointment tracking
- Patient medical history

The application supports multiple user roles including:

- Patient
- Doctor
- Receptionist
- Admin

# Features

## Authentication
- User registration
- User login
- Protected routes
- Role-based access

## Doctor Slot Management
- Create consultation slots
- Add specialization
- Set consultation fee
- Set patient limit
- Prevent expired slot booking

## Appointment Management
- Book appointments
- Prevent duplicate booking
- Prevent fully booked slots
- Update appointment status
- Appointment history tracking

## Patient Records
- Add diagnosis
- Add prescriptions
- Add consultation notes
- Consultation history management

## Dashboard Analytics
- Total appointments
- Completed consultations
- Total doctors
- Total patients

## Search & Filters
- Filter by appointment status
- Filter by specialization
- Search by patient name

## UI/UX Features
- Responsive layout
- Loading states
- Error states
- Empty states
- Status badges
- Pagination

## PDF Reports
- Download patient consultation reports
- Generate appointment reports
